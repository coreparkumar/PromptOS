# PromptOS Stack

> Project Note: This context is defined for project `PromptOS`.

## Frontend
- Electron desktop shell
- Preload bridge for safe renderer capabilities
- HTML, CSS, and JavaScript frontend

## Backend
- FastAPI application
- Python services for prompting, validation, storage, and exports
- Ollama for local model execution

## Packaging
- Electron Builder for desktop packaging
- PyInstaller for backend executable packaging when needed

## Operating Style
- Local-first
- Windows-friendly workflows
- Structured content generation
- Service-oriented background processing support
